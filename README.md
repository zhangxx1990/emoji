#### 程序员都爱 github，而许多程序员喜欢在 github 提交代码时加入 emoji 表情。不是程序员喜欢故意卖萌，而是添加了 emoji 表情的提交记录真的能包含很多有用信息，增强阅读体验。
#### 但是，emoji 表情在提交代码的时候也不能乱用，否则容易造成误解。因此开源项目&nbsp;&nbsp;专门规定了在 github 提交代码时应当遵循的 emoji 规范：gitmoji
